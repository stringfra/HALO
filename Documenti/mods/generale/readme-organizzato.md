# Documenti Organizzati Per Modifica

Questa cartella rende la documentazione facilmente esplorabile con file organizzati per modifica e fase.

## Regola di organizzazione

- `MODIFICA-XX-*` = stream di lavoro/modifica
- all'interno:
  - `00-PIANO-E-CONTESTO` = piani, analisi, documenti di contesto
  - `FASE-YY` = documenti di esecuzione fase per fase
- i file sono stati spostati fisicamente nella nuova struttura
